# Banking Financial Health Regression Model

## Objective
This project classifies the financial condition of a new bank using two ratios used in the financial analysis of banks: 
- TotLns&Lses/Assets is the ratio of total loans and leases to total assets
- TotExp/Assets is the ratio of total expenses to total assets.


## Data
The data includes data on a sample of 20 banks. 

## Content
- Intro
- Regression Model
- Estimated Equations
- Classification of financial condition of new bank
